lcg-annotate(1)
===============

NAME
----
lcg-annotate - Add comments to an existing HDF5 file generated by LCG.

SYNOPSIS
--------
[verse]
'lcg annotate' [-h | --help] [-f | --file <filename>] [-m | --message <comment>]

DESCRIPTION
-----------
'lcg-annotate' can be used to add comments to an existing HDF5 that has been previously
generated by another LCG command, most likely 'lcg-experiment'.

OPTIONS
-------
-h::
--help::
        Print a help message.

-f <filename>::
--file <filename>::
        Specify the path of the HDF5 file to which the comment will be added. If this
        option is not specified, 'lcg-annotate' will ask the user for the file name.

-m <comment>::
--message <comment>::
        The comment to add to the HDF5 file. If this option is not specified, 'lcg-annotate'
        will ask the user to enter a comment.

EXAMPLES
--------
The following command can be used to add a comment to an existing HDF5 file.

   $ lcg annotate -f some_experiment.h5 -m "The data in this file is meaningless."
